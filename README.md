# Contact Us Page

A modern and responsive **Contact Us Page** built using **React.js** and **Tailwind CSS**. This project provides users with multiple ways to connect, including support chat, phone calls, email communication, and a contact form.

## 🚀 Features

- Responsive design for desktop and mobile devices
- Clean and modern UI
- Navigation bar
- Contact information section
- Support options:
  - Via Support Chat
  - Via Call
  - Via Email
- Contact Form
  - Name Input
  - Email Input
  - Message Textarea
  - Submit Button
- Customer support illustration

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- HTML5
- CSS3

## 🎯 Learning Objectives

This project was built to practice:

- React Components
- Props
- State Management
- Event Handling
- Form Handling
- Tailwind CSS Styling
- Responsive Design Principles

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/contact-us-page.git
```

2. Navigate to the project folder

```bash
cd contact-us-page
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit

```bash
http://localhost:5173
```
